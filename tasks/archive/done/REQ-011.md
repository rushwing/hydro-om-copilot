# Goal

实现自动诊断结果的前端展示面板，周期性拉取 `/diagnosis/auto-results` 并展示 Epoch 阶段进度、待处理故障队列和已完成诊断归档。

# In Scope

- `AutoDiagnosisPanel` 组件
- Epoch 阶段条（当前自动诊断周期进度）
- 待处理队列列表（正在诊断中的故障）
- 已完成诊断结果归档列表（可展开查看详情）
- 轮询 `GET /diagnosis/auto-results` 的逻辑

# Out of Scope

- 手动诊断 UI（见 REQ-005）
- 实时推送（当前为轮询）

# Acceptance Criteria

- `AutoDiagnosisPanel` 正确展示 Epoch 阶段条
- 待处理队列与已完成列表实时更新
- 已完成诊断记录可展开查看 `root_causes`/`check_steps`/`report_draft`

# Test Case Design Notes

> 需覆盖：空队列渲染、多条诊断记录渲染、轮询失败时的降级提示。
