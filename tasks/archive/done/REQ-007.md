# Goal

实现三个基于 FastMCP 2.x 的伪传感器 Server，模拟振动摆度、调速器油压、轴承温升三类传感器数据，供 FaultAggregator 轮询使用。

# In Scope

- `vibration_sensor/server.py`：振动摆度传感器
- `governor_sensor/server.py`：调速器油压传感器
- `bearing_sensor/server.py`：轴承温升传感器
- 共享 Pydantic 模型（`shared/schemas.py`）
- 故障门限常量（`shared/thresholds.py`）
- 伪随机引擎（`shared/pseudo_random.py`）保证可重现的故障触发

# Out of Scope

- 真实传感器适配（见 REQ-019）
- FaultAggregator 聚合逻辑（见 REQ-008）

# Acceptance Criteria

- 三个 FastMCP Server 可独立启动（无互相依赖）
- 每个 Server 返回的读数符合 `shared/schemas.py` 定义的 Pydantic 模型
- 读数中故障触发率由 `thresholds.py` 门限控制
