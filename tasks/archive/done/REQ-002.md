# Goal

构建基于 LangGraph 的诊断工作流，将故障诊断分解为有序节点，支持条件路由（含图片路径），并返回完整的 AgentState。

# In Scope

- LangGraph `StateGraph` 定义，节点：`symptom_parser`、`image_agent`（条件）、`retrieval`、`reasoning`、`report_gen`
- `AgentState` TypedDict 定义（见 `backend/app/agents/state.py`）
- 条件路由：有图片时经过 `image_agent`，否则跳过
- 各节点返回 AgentState 增量更新

# Out of Scope

- RAG 检索器实现细节（见 REQ-003）
- SSE 流式事件发送（见 REQ-001）
- 知识库入库（见 REQ-004）

# Acceptance Criteria

- `StateGraph` 可通过 `graph.ainvoke()` 调用
- 无图片时经过 `symptom_parser → retrieval → reasoning → report_gen` 四节点
- 有图片时额外经过 `image_agent` 节点
- 返回的 AgentState 包含 `root_causes`、`check_steps`、`report_draft` 字段
