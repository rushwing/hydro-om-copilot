# Goal

为前端建立完整测试基础设施，支持 vitest 单元测试和 Playwright E2E 测试，为后续功能开发提供测试覆盖能力。

# In Scope

- vitest 配置（`frontend/vite.config.ts` 或独立 `vitest.config.ts`）
- `npm test` 脚本运行 vitest
- Playwright 安装与配置（`playwright.config.ts`）
- `npm run e2e` 脚本运行 Playwright
- 至少一个 vitest 示例测试和一个 Playwright 示例测试

# Out of Scope

- 具体业务测试用例（各功能 REQ 的 TC 文档负责）
- 后端测试（pytest 已就绪）
- CI 集成（见 REQ-012）

# Acceptance Criteria

- `npm test` 在 `frontend/` 目录下可运行 vitest 单元测试并输出结果
- `npm run e2e` 可运行 Playwright E2E 测试
- 配置文件存在于 `frontend/` 目录

# Test Case Design Notes

> 需覆盖：vitest 示例测试通过、Playwright 示例测试通过、`npm test` 失败时的错误输出格式。
