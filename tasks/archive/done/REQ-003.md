# Goal

实现融合 BM25 稀疏检索与 Dense 向量检索的混合检索器，使用 RRF（Reciprocal Rank Fusion）合并结果，并通过 BGE-reranker 精排，提升诊断的检索质量。

# In Scope

- BM25 稀疏检索（基于 jieba 分词）
- Dense 向量检索（ChromaDB + BAAI/bge-large-zh-v1.5 嵌入）
- RRF 融合算法，合并两路结果
- BGE-reranker-v2-m3 重排序
- 返回 Top-K 文档列表

# Out of Scope

- 知识库入库流水线（见 REQ-004）
- Qdrant 生产环境适配（Phase-005 范围）
- 检索结果溯源展示（见 REQ-015）

# Acceptance Criteria

- `HybridRetriever.retrieve(query, k)` 返回 BM25+Dense RRF 合并结果，文档数量 = k
- BGE-reranker 对合并结果重排后返回最终 Top-K，且重排前后顺序可不同（验证重排有效运行）
- 空知识库时返回空列表而非抛出异常
