# Goal

提供一键式知识库入库脚本，将 Markdown 格式的运维知识文档解析、分块、嵌入，并写入 ChromaDB 向量存储与 BM25 索引，供 HybridRetriever 查询。

# In Scope

- `scripts/ingest_kb.py` 入口脚本
- Markdown 文档解析与分块（按 frontmatter schema）
- BAAI/bge-large-zh-v1.5 嵌入生成
- ChromaDB 写入
- BM25 索引构建与持久化

# Out of Scope

- 实时增量入库（当前为全量重建）
- Qdrant 生产环境写入（Phase-005 范围）
- 文档质量校验 CI（见 REQ-012）

# Acceptance Criteria

- `python scripts/ingest_kb.py` 执行无报错
- ChromaDB 集合中可查到新入库文档
- BM25 索引中包含新入库文档
- 重复执行不产生重复文档
