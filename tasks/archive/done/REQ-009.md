# Goal

实现后台诊断 Worker，消费 FaultAggregator 产生的故障队列，自动调用 LangGraph 诊断 Pipeline 并将结果写入 DiagnosisStore。

# In Scope

- `auto_diagnosis.py` 中的 `AutoDiagnosisService`
- 故障队列消费循环
- 自动诊断专用 LangGraph：在手动诊断 Pipeline 前插入 `sensor_reader` 节点（`build_auto_graph()`）
- 调用 `graph.ainvoke()` 执行诊断
- 冷却期检查（`DIAGNOSIS_COOLDOWN` 环境变量）
- 结果写入 `diagnosis_store`（环形缓冲区）

# Out of Scope

- 故障聚合（见 REQ-008）
- 结果 API 暴露（见 REQ-010）

# Acceptance Criteria

- Worker 消费故障队列并调用 `graph.ainvoke()`
- 同一 `unit_id` 在 `DIAGNOSIS_COOLDOWN` 秒内不重复触发诊断
- 诊断结果写入 `diagnosis_store` 环形缓冲区
