# Goal

设计并实现真实传感器数据适配层，通过与 FastMCP 伪传感器相同的工具接口接入真实设备数据，使 FaultAggregator 可在不修改自身代码的情况下切换数据源。

# In Scope

- 传感器适配层接口定义（抽象基类或协议）
- 真实传感器客户端实现（MODBUS/OPC-UA/厂商 SDK 接入示例）
- 通过配置切换伪传感器与真实传感器（`SENSOR_BACKEND` 环境变量）
- FaultAggregator 零修改即可使用真实传感器

# Out of Scope

- 具体厂商 SDK 全量实现（提供接口定义和示例）
- 机组在线状态展示（见 REQ-020）
- 伪传感器 Server 修改（见 REQ-007）

# Acceptance Criteria

- 适配层接口与 FastMCP 工具接口兼容
- `SENSOR_BACKEND=real` 时 FaultAggregator 使用真实传感器适配层
- FaultAggregator 代码无需修改即可切换

# Test Case Design Notes

> 需覆盖：伪传感器模式正常工作、真实传感器模拟接入、切换后 FaultAggregator 行为一致。
