# Goal

在前端展示各机组传感器连接状态面板，实时反映在线/离线/故障状态，并在断连时给出降级提示。

# In Scope

- 机组状态面板组件
- 轮询或 SSE 订阅机组连接状态 API
- 状态变更 3 秒内更新显示
- 断连时显示降级提示（不影响已缓存的历史诊断）
- 支持多机组同时展示

# Out of Scope

- 真实传感器适配（见 REQ-019）
- 告警推送（Phase-006 后续）

# Acceptance Criteria

- 状态面板展示各机组当前连接状态
- 连接状态变更后 3 秒内 UI 更新
- 传感器断连时显示降级提示横幅

# Test Case Design Notes

> 需覆盖：正常在线状态展示、断连状态展示与降级提示、多机组同时展示。

# Agent Notes

依赖 REQ-019 适配层提供状态查询接口。
