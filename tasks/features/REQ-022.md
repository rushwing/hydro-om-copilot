# Goal

提供管理员专用的租户管理界面，支持新租户创建、配置和停用，消除直接操作数据库的需要。

# In Scope

- 租户列表页面（管理员专用路由）
- 新建租户表单（租户名称、管理员账号）
- 租户启用/停用操作
- 调用后端租户管理 API

# Out of Scope

- 多租户数据隔离后端（见 REQ-021）
- 租户级别的计量/计费

# Acceptance Criteria

- 管理员登录后可访问租户管理页面
- 通过 UI 创建新租户后租户在系统中生效
- 非管理员用户无法访问租户管理页面

# Test Case Design Notes

> 需覆盖：管理员创建租户成功、非管理员访问被拒绝、租户停用后登录被阻断。

# Agent Notes

依赖 REQ-021 后端多租户隔离就绪。管理员权限从 JWT token 的 `role` 字段识别。
