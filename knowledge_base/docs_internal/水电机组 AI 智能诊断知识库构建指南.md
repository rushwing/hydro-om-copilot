# 一、 故障机理分析库 (Failure Mechanism Analysis)

故障机理是 AI 理解“为什么会发生异常”的底层物理逻辑。

1. ## **振动与摆度机理**：

* **水力失稳机理**：当机组偏离最优工况（如 40%-60% 出力）时，转轮出口流场会形成偏心涡带，产生低频压力脉动；在高水头或大负荷下，则易发生气泡破裂引发的气蚀冲击；此外，转轮叶片与导叶间的压力场突变（动静干涉 RSI）也会产生强烈的激振力。  
* **机械不平衡与不对中机理**：转子质量中心与几何中心偏移会产生与转速平方成正比的离心力（质量不平衡）；发电机轴与水轮机轴联轴器处若存在倾斜或偏心（不对中），在转动中会产生周期性径向力试图纠正偏差，导致强烈的径向摆度。  
* **电磁干涉机理**：定转子气隙不均匀或转子绕组短路，会导致旋转过程中受到交变的磁拉力（单边不平衡磁拉力），该力随励磁电流增加而显著增强。

2. ## **轴承温升机理**：

* **换热失效机理**：冷却水流量不足导致冷却器内流体雷诺数降低、换热系数衰减；或铜管内壁结垢（如生物淤积）形成隔热层，导致热量无法有效从油侧转移到水侧，引发瓦温和油温同步飙升。  
* **油膜破裂与摩擦机理**：润滑油质恶化（乳化、含杂质）或机组低转速运行时间过长，导致无法建立动压油膜；推力轴承支撑弹簧疲劳塑变会导致载荷分配不均，引发局部干摩擦 。

3. ## **调速器油压异常机理**：

* **储能流失机理**：活塞式蓄能器密封圈磨损导致氮气进入压力油中，或压力油罐自动补气装置失灵，使得系统缓冲容量严重不足，导叶动作时油压瞬时突降。  
* **控制失效机理**：主配压阀因油质不洁（杂质卡阻）导致液压平衡受损，产生严重内漏，破坏静压支撑的稳定性，诱发系统高频抽动与油泵频繁启停。

# 二、 多参数联动逻辑库 (Multi-Parameter Linkage Logic)

AI 诊断不能依赖单一参数报警，必须建立“特征工况-伴随参数”的交叉验证逻辑矩阵。

1. ## **振动异常的联动判别**：

* **振动 \+ 负荷 (P)**：若振动仅在特定部分负荷区出现，且随开度增加而减小  → 判定为水力激振（尾水管涡带）；若振动随负荷线性增加  → 判定为机械刚度或对中问题。  
* **振动 \+ 励磁电流**：空载状态下加励磁后振动突增，且断开励磁瞬间消失  → 判定为定转子气隙不均或电气磁拉力不平衡。  
* **上机架水平振动 \+ 上导摆度**：若上机架 X 向振动远大于 Y 向（或反之），且绝对摆度正常，主要为转频成分  →判定为机架该方位支撑件松动。

2. ## **冷却与润滑异常的联动判别**：

* **瓦温升高 \+ 油温升高 \+ 冷却水温差缩小**  → 判定为冷却器结垢或换热效率劣化。  
* **瓦温升高 \+ 油温升高 \+ 冷却水流量突降/压差增大** → 判定为供水管路或滤水器堵塞。  
* **局部单块瓦温飙升 \+ 对应方位摆度增大 \+ 整体油温平稳** →  判定为轴系机械摩擦、受力不均或绝缘破坏引发轴电流电蚀。  
* **温度数据垂直阶跃跳变 \+ 伴随参数平稳** →  判定为测温元件 (PT100) 断线或仪控回路虚假信号。

# 三、 频谱特征模型库 (Spectral Characteristics)

利用大轴轴心轨迹和快速傅里叶变换（FFT），为 AI 提取确凿的频域指纹：

1. **1倍频 / 转频 (1x)**：绝对主导的 1x 分量，且幅值随转速平方成正比，相位稳定 →指向**转子质量不平衡**（动不平衡）。若 1x 伴随加励磁而突变，则指向**磁拉力不平衡** 。  
2. **2倍频 (2x)**：主频为 2x，轴心轨迹呈长椭圆状或“8”字形，且空间轴线图显示明显几何偏心  → 指向**轴系不对中**或大轴热态弯曲。  
3. **低频分量 (0.2\~0.4倍转频)**：出现大幅值低频脉动，且伴随强烈的尾水管压力脉动  → 指向**尾水管涡带激振** 1。若低频极度活跃且导致轴承油膜失稳，则指向**油膜自激振荡 (Oil Whirl)** 。  
4. **高频及叶片通过频率 (*fb*  \= *Z* ⋅ *fn*)**：频率与转轮叶片数严格对应，集中在顶盖和水导处  → 指向**动静干涉 (RSI)** 。  
5. **100Hz 工频倍频**：伴随强烈的电磁噪声 →  指向**电气不平衡/定转子摩擦** 。  
6. **全频带杂波/丰富谐波**：从低频到高频各类成分密集存在，且伴有极高频摩擦音 →  指向**转动部件与固定部件碰摩**（如油挡摩擦、止漏环摩擦）。  
7. **50Hz 纯工频**：波形呈现完美的正弦波叠加 50Hz，通常与实际转速不符 → 判定为**传感器屏蔽线破损/接地不良引入的工频干扰**。

# 四、 推理树与健康评估算法模型 (Inference Tree Models)

推理机是 AI 诊断 Agent 的“大脑”，通过正反向混合推理机制及数学模型输出结论：

1. ## **专家规则正向推理树 (If-Then Logic)**：

* *节点1*：检测到调速器油压下降速率  \>0.1MPa/s。  
* *节点2*：判断回油箱液位状态。若液位显著升高 → 判定为主配压阀内漏或蓄能器跑气；若液位正常 → 判定为外部高压管路爆裂或油泵卸载阀卡阻。  
* *输出动作*：触发二级预警，给出“核查补气电磁阀”或“立即切备用泵”的具体指令。

2. ## **设备状态欧氏距离偏离模型 (健康度量化)**：

* 建立机组在健康状态下的多维特征向量基准（负荷、水头、振动幅值、温度）。  
* 利用公式计算实时偏离度：  *d*\=∑*i*\=1*n*(*xi*−*yi*)2 （其中 *xi*为当前多源监测参数，*yi* 为历史同工况标杆数据）。当偏离度 *d* 越过动态阈值，即使未达规程报警线，Agent 也可提前发出“早期劣化预警”。

3. ## **特定设备诊断微模型 (如变压器油色谱分析)**：

* 提取主变压器油中特征气体（C2H2、C2H4、CH4、H2）。  
* 应用三比值法编码：若 C2H2/C2H4、CH4/H2、C2H4/C2H6 比值编码组合为 001，推理机直接输出“存在低于 150℃ 的低温过热故障”；若编码为 021，则输出“存在 300℃-700℃ 的高温过热故障” 30。

通过将上述四大维度的内容进行知识图谱化（Knowledge Graph），AI Agent 不仅能识别表象告警，更能沿着“工况筛选 \-\> 频谱验证 \-\> 参量交叉 \-\> 根因锁定”的专家推理树，为水电运维提供具有物理依据和规程支撑的自动诊断报告。
