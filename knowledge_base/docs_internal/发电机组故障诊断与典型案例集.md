# 发电机组故障诊断与典型案例集

当前资料的分类识别如下：

1. **更适合做 AI 诊断知识库的内容**：包含故障机理分析、多参数联动逻辑、频谱特征和推理树模型的内容。例如《水电机组典型异常诊断与处置知识库》系列报告中关于振动与摆度的联动参数分析、轴承过热的根因判别矩阵、油压下降的诊断逻辑；以及《抽水蓄能发电机组状态监测与诊断技术规范》中提及的正反向混合推理机制、各种不平衡（转子质量、磁拉力、水力）和部件摩擦的诊断模型与专家诊断逻辑。  
2. **更适合做案例库的内容**：包含具体电站或机组发生的真实故障记录、处理过程及经验总结的内容。例如《汽轮机调速系统错油门故障分析及防范》中的NG系列汽轮机错油门故障案例；《水轮发电机导轴承冷却系统探究》中的金岩水电站、株树桥水电厂烧瓦案例；以及诊断研究报告中列举的“典型案例分析与经验反馈”（如生物淤积、弹簧变形、电源消失、滤网堵塞等）。  
3. **更适合做规则库的内容**：包含国家标准、行业导则中明确的参数阈值、操作规范、技术参数标准及管理制度的内容。例如《GB/T 8564-2023 水轮发电机组安装技术规范》中的各项安装允许偏差和试验标准；《小水电技术导则》中的巡检维护周期、温度压力限制；《农村水电站安全运行与检修规程》中的温度限制、启动停机条件；以及《防止电力生产事故的二十五项重点要求》中的反事故强制措施。
