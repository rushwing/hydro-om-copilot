# 检修履历模板

## 字段模板

| 参数名称 | 示例值 | 字段说明 |
| --- | --- | --- |
| 检修记录编号 |  | 唯一标识，如：MNT-2024-0035 |
| 电站名称 |  |  |
| 机组号 |  |  |
| 检修类别 |  | A修（大修）/ B修（小修）/ 临修 / 抢修 |
| 检修日期 |  | 格式：YYYY-MM-DD |
| 设备对象 |  | 同台账模板中的设备对象名称，保持一致 |
| 发现问题 |  | 检修中发现的问题，对应故障现象描述 |
| 更换部件 |  | 部件名称及规格，多个用分号分隔 |
| 调整项 |  | 对中 / 间隙 / 配重 / 保护定值等（具体数值填入关联定值模板） |
| 验收结论 |  | 合格 / 带缺陷消缺后合格 / 不合格 |
| 关联试验报告 |  | 报告编号或文件名 |
| 责任班组 |  | 执行单位或班组名称 |

## 备注

- 对“检修后首次并网即异常”这类场景，检修履历对根因排序权重应显著提高。
