# 历次缺陷模板

## 字段模板

| 参数名称 | 示例值 | 字段说明 |
| --- | --- | --- |
| 缺陷编号 |  | 唯一标识，如：DEF-2024-0012 |
| 电站名称 |  |  |
| 机组号 |  |  |
| 设备对象 |  | 同台账模板中的设备对象名称，保持一致 |
| 故障现象 |  | 可观察到的异常表现，客观描述 |
| 首次发现日期 |  | 格式：YYYY-MM-DD |
| 当前状态 |  | 待处理 / 已处理 / 观察中 |
| 严重程度 |  | 轻微(minor) / 一般(normal) / 重要(major) / 致命(critical) |
| 关联测点 |  | tag_code，多个用逗号分隔 |
| 关联报警 |  | 报警名称或编号 |
| 临时处置 |  | 快速止损操作，未必消除根因 |
| 根因 |  | 经分析确认的最终原因 |
| 根治措施 |  | 消除根因的永久性措施 |
| 复发情况 |  | 是 / 否；若是，填写复发日期 |

## 备注

- 缺陷历史是构建“相似机组、相似部位、相似季节、相似工况”检索能力的关键现场层数据。
