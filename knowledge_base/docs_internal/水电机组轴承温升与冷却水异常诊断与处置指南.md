# 水电机组轴承温升与冷却水异常诊断与处置指南

内容基于设备状态监测导则、机组运行规程及实际故障案例提炼。

# 一、诊断框架

## 1\. 四大轴承的常见温升场景

* **上导轴承**：由于靠近发电机顶部，其温升多与发电机内部的冷热风循环受阻、转子质量不平衡带来的径向负荷、以及轴承座绝缘破损引发的轴电流电蚀发热相关。  
* **下导轴承**：受发电机下机架刚度及机械摆度的直接影响，温升多与大轴径向不对中或机架支撑件松动有关。  
* **推力轴承**：作为承载机组全部转动部分重量和巨大水推力的核心，其温升逻辑与负荷变化（水头、有功功率）呈现高度相关性；受力不均极易导致单块瓦温飙升。  
* **水导轴承**：因贴近水轮机顶盖，不仅承受转轮侧的水力振动（如涡带激振），还容易受到水质泥沙磨损、顶盖漏水及主轴密封损坏的直接影响。

## 2\. 异常原因区分逻辑

* **冷却水流量不足**：表现为冷却水压力突降或滤水器压差增大，冷却水出口温度显著升高；由于冷却动力不足，油温与瓦温同步且快速上升。  
* **冷却器换热下降**：表现为“流量和压力充足，但进出水温差异常缩小”。这是由于冷却器铜管内壁结垢（如生物淤积）形成了隔热层，热量无法有效从油侧传导至水侧，导致油温偏高。  
* **油循环异常**：多见于自循环轴承，因环境温度过低（如低于 10℃）导致油粘度过大，或油槽内部挡油板损坏，热油无法有效流向冷却器，形成局部“热死区”，表现为瓦温高但冷却器出水温度不高。  
* **测温元件异常**：表现为温度数值发生逻辑不可达的垂直阶跃跳变（如瞬间从 45℃ 变为 0℃ 或 150℃），而相邻瓦温、油温、振动等伴随参数均保持稳定，通常为 PT100 断线或信号干扰。

# 二、参数与判据

当出现轴承温升或冷却水异常时，应通过构建以下参数的联动矩阵来精准定位问题：

| 监测参数 | 诊断意义与判据 | 定位指向 |
| :---: | :---: | :---: |
| **轴承温度** | 反映机械摩擦热平衡。报警阈值一般为 60-65℃，跳闸阈值为 70℃。 | 若单块瓦温突升，指向局部受力过载或摩擦；若整体同步上升，指向冷却或润滑共性失效。 |
| **油温** | 反映润滑油的整体散热基础。正常区间为 5-55℃。 | 若油温升幅落后于瓦温，说明热量产生极快但传导慢，高度疑似干摩擦或严重偏磨。 |
| **冷却水进出口温度** | 反映冷却器的换热效率。正常温差应在 2-5℃。 | 温差变小预示铜管结垢或油侧不循环；温差异常增大预示热负荷严重过载。 |
| **冷却水流量/压力** | 反映冷却系统动力。压力需维持在 0.15-0.30MPa 以上。 | 压力和流量突降，直接指向供水中断或滤水器严重堵塞。 |
| **油位** | 反映油槽介质的完整性。 | 油位突降预示管路漏油；油位上升且油色乳化说明冷却器铜管破裂漏水入油。 |
| **油压/油流** | 反映强迫油循环系统的泵组状态。 | 油流指示器不动作或油压跌落，预示循环泵故障或管路气阻。 |
| **振动/摆度** | 反映机械稳定性与轴心轨迹。 | 若瓦温升高伴随摆度突增，直接判定为机械部件动静摩擦或严重不对中。 |
| **负荷 (P/H)** | 温升的基准因子。水头和有功越大，自然温升基准越高。 | 用于建立工况-温度基准线，若负荷未变而温度爬升，则为异常劣化。 |

# 三、根因分类

| 根因分类 | 典型症状 | 确认方法 | 处置建议 |
| ----- | ----- | ----- | ----- |
| **冷却系统问题** | 多块瓦温与油温同步单调上升，进出水温差缩小，滤水器前后压差增大。 | 现场核实供水阀门开度，检查滤水器排污是否动作，测进出口水压。 | 立即切换至备用滤水器，开大旁路供水阀；无法控制时主动降负荷。 |
| **润滑系统问题** | 油槽内有泡沫、油色浑浊乳化（进水），油位异常，局部瓦温突升。 | 现场取样进行油质化验（水分、杂质），排查冷却器是否漏水。 | 启用在线滤油机除水脱渣，补充同型号新油；严重劣化需停机换油。 |
| **机械摩擦或轴承本体问题** | 局部单块瓦温飙升（不对称），机组振动及摆度超标，停机惰走时间明显缩短。 | 检查推力瓦弹性油箱压力，做轴系特性测试，红外测温轴承座分布。 | 若伴随剧烈振动需立即手动急停，解体检查轴瓦受力平衡并重新研刮。 |
| **工况相关问题** | 机组在特定负荷（振动区）运行时瓦温上升，或投励磁后上导瓦温异常波动。 | 查阅稳定性试验报告核对振动区；测量轴电流，检查轴承座绝缘阻值。 | 调整出力避开振动区运行；停机清理碳刷粉尘，修复轴承绝缘垫破损点。 |
| **测温/仪表问题** | 温度数值呈现无物理过程的垂直跳变，其他同组瓦温、油温、振动均平稳。 | 测量 PT100 引线电阻排查断线，现场用红外测温枪实地复核比对。 | 在监控中暂屏蔽该故障点并强制赋均值防误跳闸，加强人工巡检。 |

# 四、处置与预防

## 1\. 现场检查步骤 (SOP)

* **一核对**：立即调取历史曲线，辨别是数据“突变”（多为传感器或机械瞬发故障）还是“缓变”（多为冷却劣化），交叉比对瓦温与油温。  
* **现场三步法**：  
  * **一摸**：手摸冷却水进出管路感受温差，触摸轴承盖判断有无高频振感。  
  * **二看**：查看油位计液位、油色，观察油流指示器是否翻转，核对滤水器压差。  
  * **三听**：贴近轴承室，听诊是否有尖锐的金属摩擦声或不规则的撞击声。

## 2\. 临时处置与降负荷考虑

* **冷却干预**：第一时间清洗或切换备用滤水器，手动开大冷却水主阀或旁路阀。  
* **降负荷策略**：当瓦温达到预警值（如 65℃）时，应主动申请降低机组有功/无功负荷（如减至 50%），观察温度是否回落；若回落，证明温升与水推力/电磁力过载强相关。

## 3\. 紧急停机判断依据

* 瓦温达到保护跳闸阈值（通常 70℃）且保护未动作时，必须立即手动事故停机。  
* 现场发现推力头伴有烧焦气味、油槽冒出烟雾，或伴有剧烈金属摩擦声时，严禁继续运行。

## 4\. 后续消缺与预防措施

* **长效维护**：在汛期多沙、多杂物季节，缩短自动滤水器排污周期；定期对冷却器进行化学/物理反冲洗防结垢。  
* **精细化检修**：大修期间使用高精度仪器重新调整推力瓦受力平衡（受力偏差控制在 ±10% 内）；抽检运行 10 年以上的轴承底座接地及绝缘情况，防范电蚀。  
* **防火管理**：严格落实反措，热体附近的润滑油法兰必须加装金属保护罩，防范漏油引发火灾。

# 五、案例提取

## 1.【案例 1：冷却器生物淤积引发的群体温升】

* **故障现象**：夏季运行期间，上导、推力、下导瓦温在三周内阶梯式上升。  
* **关联参数变化**：冷却水进出口温差从 3.5℃ 降至 0.8℃，滤水器压差正常，水流量正常。  
* **根因**：冷却器铜管内壁附着大量淡水壳菜（生物淤积），形成生物隔热层，严重阻碍热交换。  
* **临时处置**：调大进水总阀增加循环水量，申请低负荷运行以降低机组产热。  
* **根治措施**：停机进行化学清洗及机械物理清理；在进水端加装超声波防生物附着装置。  
* **经验总结**：诊断系统应收录季节性生物淤积的特征，并在水温高的特定月份自动提示运维人员加强温差趋势分析。

## 2.【案例 2：推力轴承支撑力不均导致的局部烧瓦】

* **故障现象**：机组带满负荷后，推力轴承 8 号瓦温在 15 分钟内从 48℃ 飙升至 72℃，触发事故停机。  
* **关联参数变化**：仅 8 号单一瓦温异常（其余瓦温波动极小），跳闸瞬间振动摆度出现峰值。  
* **根因**：该瓦下方的支撑弹簧发生疲劳塑性变形，导致其承载压力远超其他瓦面，无法形成有效动压油膜。  
* **临时处置**：保护动作，紧急停机。  
* **根治措施**：解体检查，更换全套受损支撑元件，并重新进行推力瓦受力平衡调整（受力偏差严格控制在 ±10% 以内）。  
* **经验总结**：AI 诊断应具备“空间一致性”分析能力，通过对比同组各瓦温的差异，快速识别硬件受力缺陷。

## 3.【案例 3：油槽缺油引发的导轴承烧瓦】

* **故障现象**：某电厂 1 号水轮发电机组导轴承推力瓦烧毁，导致机组被迫紧急停机。  
* **关联参数变化**：轴承油槽油位显著下降，瓦温迅速上升突破跳闸极限。  
* **根因**：运行过程中油槽内少油、缺油，导致轴承失去了必要的润滑介质，动压油膜破裂引发干摩擦。  
* **临时处置**：立即紧急停机。  
* **根治措施**：修复或更换烧毁的导轴承及推力瓦；查明漏油点并彻底修复，重新注油至标准液位。  
* **经验总结**：必须加强对轴承油位计和油流指示器的日常巡视，将油位异常下降作为高优预警指标，严防机组缺油运行。
